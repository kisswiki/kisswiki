## Add Firefox to Big Picture Mode

First in Desktop Mode: `right-click on Firefox icon > Add to Steam`

## Maximize in fullscreen in Big Picture Mode

`Hamburger Menu > Zoom section and click on icon to Fullscreen (F11)`. Rerun Firefox because you will not be able to click.

This was the inspiration https://www.reddit.com/r/SteamDeck/comments/xh3ahf/comment/n351dqj/

## Reverse mouse wheel scrolling in Big Picture Mode

`about:config > mousewheel.default.delta_multiplier_y: -100`

https://support.mozilla.org/en-US/questions/1014510#answer-612985
