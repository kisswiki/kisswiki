## keyboard shortcuts

```
WIN + CTRL + LEFT/RIGHT: Switch to previous or next desktop
WIN + CTRL + D: Create a new desktop
WIN + CTRL + F4: Close the current desktop
WIN + TAB: Launch task view
```
