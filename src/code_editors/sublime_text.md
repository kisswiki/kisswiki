## Customize

`cmd+shift+p > Install Package Control`

`cmd+shift+p > Install Package > Inspired Git​Hub Color Scheme`

`Settings > Settings > "font_options": ["no_bold", "no_italic"],`
`Settings > Settings > "font_face": "Reddit Mono",`

`Settings > Customize Theme > "variables":{"ui_bg": "white",},`

- https://packagecontrol.io/packages/Inspired%20GitHub%20Color%20Scheme
- https://packagecontrol.io/packages/Visual%20Studio%20Code%20Plus%20Scheme
