## Config

Config at `~/.config/kitty/kitty.conf`:

```
font_family Consolas Ligaturized
font_size 13.0
foreground #000000
background #ffffff
tab_bar_style separator
tab_bar_edge top
tab_separator " "
inactive_tab_background #bbb
```
