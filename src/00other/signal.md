## Restore backup on Android

1. Install Signal
2. Signal > Settings > Chats > Chat backups > Turn on > select directory
3. Uninstall Signal
4. Put your backup in the directory you selected previously
5. Install Signal
6. Continue, Continue
7. Restore backup
