## git

```bash
apt-add-repository ppa:git-core/ppa
apt-get update
apt-get install git
```

## nodejs

```bash
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
```
